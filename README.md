# Real-Time Chat Application

A real-time chat application built using React, Node.js, Express, and Socket.IO. This project allows multiple users to join chat rooms and communicate instantly in real time.

---

# Features

- Real-time messaging
- Join chat rooms
- Multiple users support
- Responsive user interface
- Fast communication using Socket.IO
- Simple and clean frontend design

---

# Tech Stack

## Frontend
- React.js
- Vite
- CSS

## Backend
- Node.js
- Express.js
- Socket.IO

## Database
- MongoDB (Optional)

**How It Works**

-User joins a chat room
-Frontend sends data to backend using Socket.IO
-Backend broadcasts messages in real time
-All connected users receive messages instantly
